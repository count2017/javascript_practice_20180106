【arguments】 JavaScript还有一个免费赠送的关键字arguments， 它只在函数内部起作用，并且永远指向当前函数的调用者传入的所有参数。 arguments类似Array但它不是一个Array：

实际上arguments最常用于判断传入参数的个数。你可能会看到这样的写法：
// foo(a[, b], c)
// 接收2~3个参数，b是可选参数，如果只传2个参数，b默认为null：
function foo(a, b, c) {
if (arguments.length === 2) {
// 实际拿到的参数是a和b，c为undefined
c = b; // 把b赋给c
b = null; // b变为默认值
}
// ...
}
